# Desafio de Testes - Fin Api

A Fin Api é uma aplicação onde é possível fazer depósitos, retiradas e consultas em uma conta.

<br />

## 🧩 Desafio

Esse é um desafio da trilha de Node.js do programa Ignite oferecido pela [Rocketseat](https://rocketseat.com.br).

Onde tive que fazer a construção de testes unitários e testes de integração.

## 🛠️ Tecnologias e Ferramentas

- Node.js
- Typescript
- Express
- Docker
- Jest
- Typeorm
- Postgres
